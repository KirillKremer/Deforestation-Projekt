# Deforestation Analysis (SQL)

## Beschreibung
In diesem Projekt habe ich Daten zur globalen Entwaldung analysiert. Ziel war es, Veränderungen der Waldflächen weltweit zu untersuchen und Unterschiede zwischen Regionen und Ländern zu erkennen.

## Technologien
- SQL (PostgreSQL)

## Datengrundlage
Die Analyse basiert auf mehreren Datensätzen:
- forest_area.csv
- land_area.csv
- regions.csv

Diese wurden mithilfe von JOINs zu einer gemeinsamen Datenbasis zusammengeführt.

## Analyse
Die SQL-Queries beantworten unter anderem folgende Fragestellungen:
- Wie hat sich die globale Waldfläche zwischen 1990 und 2016 entwickelt?
- Welche Regionen sind besonders stark betroffen?
- Welche Länder haben die größten Veränderungen der Waldfläche?

Zur besseren Auswertung wurde zusätzlich eine View erstellt, die die Daten kombiniert und vereinfacht analysierbar macht.

## Ergebnis
Durch die Analyse konnten Trends der globalen Entwaldung sowie deutliche Unterschiede zwischen Regionen und einzelnen Ländern identifiziert werden.

## Nutzung
Die SQL-Datei enthält alle verwendeten Queries.

## Quelle
Das Projekt basiert auf einer Aufgabenstellung aus einem Udacity-Kurs.  
Die Analyse und Umsetzung der SQL-Queries wurden eigenständig durchgeführt.
